# ReminderLogin

แอปเล็ก ๆ สำหรับช่วยเตือนว่าคุณได้ล็อคอินเกมโปรดในวันนี้แล้วหรือยัง โดยทำงานเป็นหน้าต่างลอยเล็ก ๆ อยู่มุมขวาบนของเดสก์ท็อป พร้อมช่องให้ติ๊กว่าเกมไหนล็อคอินเสร็จแล้วบ้าง

## คุณสมบัติหลัก

- เลือกเองได้ว่าจะให้เตือนเกมอะไรบ้าง พร้อมระบุไฟล์โปรแกรมของเกมเพื่อให้ระบบติ๊กอัตโนมัติเมื่อเปิดเกมนั้น
- เพิ่มชื่อเกมใหม่ ๆ ได้ตามต้องการ
- บันทึกสถานะการล็อคอินไว้แบบรายวัน และรีเซ็ตให้อัตโนมัติเมื่อเข้าสู่วันใหม่
- ตั้งค่าให้เปิดขึ้นเองอัตโนมัติทุกครั้งที่เข้าสู่ระบบได้โดยตรงจากแอป (รองรับ Windows, macOS และ Linux เดสก์ท็อปที่ใช้มาตรฐาน XDG autostart)
- หน้าต่างแอปอยู่ด้านบนสุด (always-on-top) และจัดตำแหน่งไว้มุมขวาบนโดยอัตโนมัติ

## การใช้งาน

1. ติดตั้ง Python 3.8 ขึ้นไป (มาพร้อม Tkinter อยู่แล้วในชุดติดตั้งมาตรฐาน)
2. รันโปรแกรมด้วยคำสั่ง

   ```bash
   python main.py
   ```

   หรือจะรันแบบโมดูลก็ได้เช่น

   ```bash
   python -m reminder_login
   ```

3. ถ้าไม่อยากเปิดเทอร์มินัลค้างไว้ สามารถสั่งให้แอปรันแบบฉากหลังด้วยคำสั่ง

   ```bash
   python -m reminder_login --background
   ```

   คำสั่งนี้จะสร้างโปรเซสใหม่แล้วคืนคอนโทรลให้เทอร์มินัลทันที คุณจึงปิดหน้าต่างเทอร์มินัลได้เลยโดยที่ตัวแอปยังทำงานอยู่ หรือหากใช้ Windows/Mac ที่ผูกไฟล์ `.pyw` กับ `pythonw.exe` อยู่แล้ว สามารถดับเบิลคลิกไฟล์ `launch_reminder_login.pyw` ที่อยู่ในโฟลเดอร์โปรเจ็กต์เพื่อเปิดแอปโดยไม่ต้องผ่านเทอร์มินัลเลยก็ได้

4. หน้าต่างจะลอยอยู่มุมขวาบนของหน้าจอ
   - ติ๊กถูกหน้าเกมที่ล็อคอินแล้วในแต่ละวัน
   - กดปุ่ม **จัดการรายชื่อเกม** เพื่อเลือกว่าจะให้เตือนเกมไหนบ้าง และกำหนดไฟล์ `.exe`/`.app`/ไบนารี่ของเกมเพื่อให้ระบบตรวจจับการเปิดเกมแล้วติ๊กให้อัตโนมัติ
   - กดปุ่ม **เพิ่มเกมใหม่** เพื่อเพิ่มชื่อเกมที่ยังไม่มีในรายการ พร้อมเลือกให้เตือนทันทีได้ และเลือกไฟล์เกมได้จากหน้าต่างนี้เลย
   - ถ้าอยากเริ่มนับใหม่สำหรับวันนั้นให้กดปุ่ม **เริ่มใหม่สำหรับวันนี้**
   - ถ้าต้องการให้แอปเปิดขึ้นเองทุกครั้งที่เข้าสู่ระบบ ให้ติ๊กตัวเลือก **เปิดเองอัตโนมัติเมื่อเข้าสู่ระบบ** (รองรับ Windows, macOS และ Linux ที่ใช้มาตรฐาน autostart)

5. หากต้องการตั้งค่า auto-start ผ่านคำสั่งเทอร์มินัล สามารถใช้ตัวเลือกดังนี้

   ```bash
   # ติดตั้งให้เปิดเองทุกครั้งที่เข้าสู่ระบบ
   python -m reminder_login --install-autostart

   # ยกเลิกการเปิดเอง
   python -m reminder_login --remove-autostart

   # ตรวจสอบสถานะปัจจุบัน
   python -m reminder_login --autostart-status
   ```

## ไฟล์การตั้งค่า

แอปจะสร้างไฟล์บันทึกข้อมูลไว้ที่โฟลเดอร์ `~/.reminder_login/state.json` (หรือ `%USERPROFILE%\.reminder_login\state.json` บน Windows) เพื่อเก็บรายการเกม สถานะการล็อคอินในแต่ละวัน และไฟล์เกมที่ใช้สำหรับตรวจจับอัตโนมัติ หากต้องการเริ่มใหม่สามารถลบไฟล์นี้ได้เมื่อปิดโปรแกรมอยู่

## หมายเหตุ

- แอปนี้สร้างจาก Tkinter ซึ่งมีอยู่แล้วใน Python มาตรฐาน ไม่จำเป็นต้องติดตั้งไลบรารีเพิ่มเติม
- หากใช้หลายหน้าจอ หน้าต่างจะอ้างอิงตำแหน่งของหน้าจอหลัก (primary display)
